# Claude Code GitHub配置文件

## 创建拉取请求
使用 "pr" 命令让Claude创建拉取请求，Claude会：
- 分析代码差异
- 生成适当的提交消息
- 创建有意义的PR描述

## 代码审查评论
告诉Claude修复PR评论：
- "修复PR上的评论"
- "根据评论意见修改代码"
- "处理代码审查反馈"

## 构建修复
Claude可以：
- 分析构建失败原因
- 修复代码检查警告
- 自动修正语法错误

## Issues管理
Claude可以：
- "循环遍历开放的GitHub issues"
- 分类和整理issues
- 创建新的issue

## 使用示例
1. 创建PR：直接说"创建一个pr"
2. 修复评论：说"修复PR上的评论"
3. 修复构建：说"修复构建失败"
4. 整理issues：说"整理open的issues"
